# Triple Espresso

## Descripción del proyecto

Este es el cuarto proyecto del programa de TripleTen, donde desarrollé una página web moderna para una cafetería. El objetivo principal fue aplicar conceptos avanzados de HTML y CSS, enfocándome en la precisión del diseño y la organización del código.

### Tecnologías y herramientas usadas:

- **HTML5**: Estructura semántica de la página.
- **CSS3**: Estilos detallados, incluyendo Flexbox para la alineación.
- **Metodología BEM**: Para mantener un código organizado y escalable.
- **Posicionamiento Absoluto**: Utilizado para elementos decorativos como los círculos del fondo.
- **GitHub Pages**: Para el despliegue y visualización en línea.

## Planes de mejora

En el futuro, me gustaría implementar las siguientes mejoras:

1. **Diseño Responsivo**: Adaptar la página para que se vea perfecta en dispositivos móviles y tablets.
2. **Formulario funcional**: Conectar el formulario de reservaciones con un servicio de backend o correo electrónico para recibir las solicitudes reales.
3. **Animaciones**: Agregar micro-interacciones en los botones y videos de recetas para una mejor experiencia de usuario.
